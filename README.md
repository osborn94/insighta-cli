# Insighta CLI

A command-line interface for interacting with the Insighta Labs Profile Intelligence API.

This CLI enables authenticated users to query, filter, analyze, and export demographic profile data directly from the terminal.

---

## Overview

The Insighta CLI is part of the Insighta Labs+ platform. It provides:

- Secure authentication via GitHub OAuth (PKCE)
- Token-based API access
- Advanced querying and filtering of profiles
- Natural language search capabilities
- CSV export functionality

All commands communicate with a centralized backend service.

---

## Tech Stack

- Node.js
- Commander.js
- Axios
- GitHub OAuth (PKCE)
- File-based credential storage
